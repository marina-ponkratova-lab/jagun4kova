# Weather Diary

**Автор:** Джагунчикова Алина

## Описание программы

Weather Diary — простое приложение для ведения дневника погоды с графическим интерфейсом. Позволяет добавлять записи о погоде (дата, температура, описание и осадки), просматривать их в таблице и фильтровать по дате или температуре. Все данные сохраняются в формате JSON.

## Как использовать

1. Установите Python 3.x.
2. Скопируйте файлы проекта в одну папку.
3. Запустите main.py.
4. Введите дату (ГГГГ-ММ-ДД), температуру (число), описание погоды и отметьте осадки при необходимости.
5. Нажмите «Добавить запись».
6. Для фильтрации используйте вкладку «Фильтр».
7. Данные сохраняются автоматически.

## Примеры использования

- Добавить запись: 2026-04-24; +22; ясно; осадков нет.
- Отфильтровать записи за 2026-04-24.
- Показать только записи с температурой выше +15°C.
